ChemAgent: Self-updating Library in Large Language Models Improves Chemical Reasoning

Xiangru Tang, Tianyu Hu, Muyang Ye, Yanjun Shao, Xunjian Yin, Siru Ouyang, Wangchunshu Zhou, Pan Lu, Zhuosheng Zhang, Yilun Zhao, Arman Cohan, Mark Gerstein

ICLR 2025

Paper Code

The Chemistry Challenge

Chemistry presents a unique challenge for language models. It requires not just general reasoning ability, but deep domain knowledge—reaction mechanisms, molecular properties, thermodynamic principles. Most LLMs weren't trained with enough chemistry-specific data to handle these tasks reliably.

The traditional solutions—fine-tuning on chemistry datasets or using static knowledge bases—have limitations. Fine-tuning is expensive and doesn't scale across domains. Static knowledge bases can't adapt to new problems or learn from experience.

A Library That Grows

ChemAgent takes a different approach: equip the model with a self-updating library. As the agent solves chemistry problems, it doesn't just produce answers—it accumulates knowledge. Successful reasoning patterns, useful chemical facts, and problem-solving strategies are stored in a structured library for future use.

When the agent encounters a new problem, it retrieves relevant knowledge from this library to enhance its reasoning. The library grows and improves with each interaction, becoming more valuable over time.

The Key Innovation

Unlike static retrieval augmentation, ChemAgent's library is dynamic. It learns from experience without requiring parameter updates. The agent effectively builds its own chemistry textbook through problem-solving, capturing not just facts but the reasoning patterns that make those facts useful.

No Fine-Tuning Required

Perhaps most remarkably, ChemAgent works with off-the-shelf language models. There's no domain-specific training, no expensive fine-tuning on chemistry datasets. The model bootstraps its chemistry expertise through the self-updating library mechanism, starting from general-purpose capabilities and specializing through experience.

This makes the approach highly practical—it can be applied to any LLM and extended to other domains beyond chemistry.

Key Results

  • SciBench Performance: Significant improvements over baseline LLMs and existing augmentation methods
  • Knowledge Accumulation: Library quality improves with more problem-solving interactions
  • Transferability: Knowledge learned from one set of problems transfers to related problems
  • Interpretability: The library provides insights into the agent's reasoning process

Learning to Learn Chemistry

ChemAgent represents a new paradigm for domain adaptation in LLMs. Instead of trying to encode all possible domain knowledge during training, we let models accumulate expertise through experience. This mirrors how human experts develop—through practice, reflection, and building personal knowledge repositories.

The implications extend beyond chemistry to any domain requiring specialized knowledge: medicine, law, engineering, and beyond.
